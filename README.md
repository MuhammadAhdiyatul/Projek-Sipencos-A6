# SiPencos — Sistem Pencari Kos Berbasis Desktop 🏠

[![Python Version](https://img.shields.io/badge/python-3.8%2B-blue.svg)](https://www.python.org/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Project Status: Active](https://img.shields.io/badge/Status-Development-green.svg)]()

**SiPencos** adalah aplikasi desktop berbasis Python yang dirancang untuk mempermudah mahasiswa dan pencari hunian dalam menemukan kos yang sesuai dengan kebutuhan. Dengan pendekatan data-driven, aplikasi ini tidak hanya menampilkan daftar kos, tetapi juga menyediakan fitur analisis untuk pengambilan keputusan yang lebih objektif.

---

## 🚀 Fitur Utama

- **Pencarian Cerdas**: Cari hunian berdasarkan lokasi, nama, atau kata kunci tertentu.
- **Filter Multi-Kriteria**: Saring hasil berdasarkan rentang harga, fasilitas (WiFi, AC, Kamar Mandi Dalam), dan durasi sewa.
- **Visualisasi Data**: Lihat statistik distribusi harga dan ketersediaan kos melalui grafik interaktif untuk membantu membandingkan nilai pasar.
- **Manajemen Data Terstruktur**: Integrasi data dari berbagai sumber yang disajikan dalam format yang rapi dan mudah dibaca.
- **Antarmuka Desktop**: Pengalaman pengguna yang responsif dan ringan menggunakan library GUI Python.

---

## 🛠️ Teknologi yang Digunakan

- **Bahasa Pemrograman**: [Python](https://www.python.org/)
- **Library GUI**: `Tkinter` / `CustomTkinter` (Menyediakan tampilan modern)
- **Pengolahan Data**: `Pandas`
- **Visualisasi**: `Matplotlib` / `Seaborn`
- **Penyimpanan Data**: `CSV` / `SQLite`
