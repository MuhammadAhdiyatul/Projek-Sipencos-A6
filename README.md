# Projek-Sipencos-A6
🏠 SiPencos — Sistem Pencari Kos Berbasis Desktop SiPencos adalah aplikasi desktop berbasis Python untuk memudahkan pencarian kos. Aplikasi ini menyediakan fitur pencarian, filter, dan visualisasi data kos dari berbagai sumber untuk membantu pengguna mengambil keputusan dengan lebih cepat dan terstruktur.
